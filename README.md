# AI Flashcard Generator App

## Week 4 Head Starter Project

This web application allows users to generate and manage flashcards, leveraging modern web technologies and services for a comprehensive learning experience.

### Key Features

- Generate 10 flashcards on demand
- Save flashcards to a Firebase database for persistent storage
- User authentication powered by Clerk
- Integration with Stripe API for future monetization possibilities

### Tech Stack

- Frontend: React, Next.js, MaterialUI
- Backend: Firebase
- Authentication: Clerk
- Payment Integration: Stripe API

### Future Plans

- Implement paid tiers for advanced features
- Expand flashcard generation capabilities
- Add more customization options for users

This project was developed as part of the Week 4 Head Starter Project curriculum.
